Returns the version number of the Royale library, including SCM revision number.

Parameters
majorThe major component of the version number
minorThe minor component of the version number
patchThe patch component of the version number
buildThe build number, if built on the release build server
scmThe source control revision that the library was built from. May be empty if the SCM revision wasn't available, and may have the postfix "-dirty" if there were local changes that weren't checked in to source control.
Returns
CameraStatus.SUCCESS, or an error code if allocating memory for the SCM revision string failed.

If the SCM number is not needed, this can also be read from the AssemblyInfo: typeof(RoyaleDotNet.CameraDevice).Assembly.GetName().Version
