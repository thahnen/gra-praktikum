Detailed Description

template<class Category, class T, class Distance, class Pointer, class Reference>
class royale::iterator::royale_base_iterator< Category, T, Distance, Pointer, Reference >

Royale base iterator class (basis for all non-const iterators)

Holding the basic functions for a non-const iterator. Other iterators can be based on this (if they follow the non-const approach), while this class itself is directly based on the skeleton.

The royale non-const forward iterator and the royale non-const reverse iterator are based on this class.
